Introduz o arquivo `autoapply_ai_dashboard.html`, servindo como a interface de usuário principal para monitorar o projeto AutoApply AI. Componentes implementados: - Design system dark mode focado em terminal (fontes monospace e grid layout). - Painel de KPIs (Vagas novas, Score médio, Candidaturas, E-mails lidos). - Tabela interativa de vagas com filtros por tecnologia (Python, Dados), modelo de trabalho e score. - Timeline visual para rastreamento de status de candidaturas. - Interface de chat simulada para interação com o agente AI e execução de comandos rápidos. - Visualizador de logs de execução do sistema.
