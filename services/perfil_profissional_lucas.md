## 4. Projetos de Destaque

### 🤖 Projeto 1: AutoApply_AI *(Em Desenvolvimento — 2026)*
**Repositório:** [github.com/lucascabralds/AutoApply_AI](https://github.com/lucascabralds/AutoApply_AI)

- **Escopo:** Pipeline inteligente de automação de e-mails que atua como assistente de carreira em segundo plano
- **Tecnologias:** Python, Gmail API (GCP), Gemini API (Google AI), MIME
- **O que faz:**
  - Monitora ativamente a caixa de entrada e extrai dados estruturados de e-mails de vagas
  - Usa IA (Gemini) para identificar empresa, cargo, requisitos técnicos e comportamentais
  - Cruza requisitos da vaga com o currículo do candidato e envia candidatura automaticamente apenas quando há match
  - Para e-mails que não são vagas, gera resumos executivos do que está pendente
- **Destaque:** Demonstra capacidade de construir um sistema end-to-end com IA aplicada, integrando APIs de cloud, processamento de linguagem natural e lógica de negócio — habilidade altamente valorizada no mercado atual
